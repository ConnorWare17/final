The early history of HTML and the Internet

Time Event
1989 Tim Berners-Lee, a CERN scientist, creates HTML to allow for easier use and transfer of documents. Using hypertext to create a "web of info to cross-reference at any given time."
1980s The HTML system worked well with the newly popular system of domain names, creating the perfect oppurtunity.
1980s HTML was very much based on a language called SGML, and it inherited many of its elements like having dual tags i.e. <tag> </tag>. Many tags were also taken stright from SGML, like the H1-6 tags, P, the list tags, and footer.
1991 Berners-Lee creates HTML+, an enhanced version of html.
1992 Implementing images proved to be rather devisive between web nerds of the time, but a man named Marc Andreesen set out to implement them regardless and the img tag he created is still in use today.
1993 This year saw the creation of the Mosaic web browser, but it saw criticism for how it was designed and how any features were poorly implemented.
1994 HTML+ showcased at the first World Wide Web conference at CERN to critical acclaim.
1994 An HTML working group is set up by the Internet Engineering Task Force, I.E.T.F.
1994 HTML 2, an effort to re-standardize HTML begins by adding many frequently used tags from other sources.
1994 Andreesen forms Netscape, a browser that exerted lots of autonomy from the rest of the HTML community by adding new stuff as it saw fit.
1994 The World Wide Web Consortium is formed in order to further innovate and standardize HTML
1995 HTML sees a massive expansion, to the point where HTML 3 is declared
1995 Microsoft creates their own browser, Internet Explorer.
1995 The Task Force falls apart, as their team just could not handle the sheer volume of emails coming in.
1995 the various browser companies mostly come together to standardize their code.
1995 Cascading Style Sheets, CSS, beome the standard for design.
1996 Work begins on HTML 4.
1998 The launch of Google.
2000 The .com bubble bursts, causing many internet and tech startups and businesses to go under.
2001 The launch of Wikipedia.
